== Description ==

Jordy is the clean minimal and easy to use WordPress theme suitable for all types of blog sites.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Jordy in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
