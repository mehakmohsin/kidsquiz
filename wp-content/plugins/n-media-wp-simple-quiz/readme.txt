WordPress Simple Quiz - MCQS Made Easy

== Description ==
Wordpress Simple Quiz is simple plugin to create a unlimited quizzes with multiple choices. Each question can have multiple choices/answers and correct answer can be set very easily with nice UI. 
Each question will have total marks. Once result is submitted user can see result optionally (PRO) and provide his information.

Results are saved as custom post type with all information like total marks, obtained marks, correct answers etc. Once result is save it can be sent to other people via email from result page in PRO version

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can create Quiz from `WP Simple Quiz` menu
